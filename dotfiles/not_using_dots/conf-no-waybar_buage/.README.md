## what is pywal?

pywal 是一個能根據你桌布顏色「自動生成」配色方案（Color Scheme）的工具。

它的工作原理是：

你給它一張圖片。
它提取圖片中的主色調。
它將這些顏色應用到終端機（Terminal）和其他支援的軟體（如 Waybar、Rofi、Vim）中。

1. 安裝 pywal：

```bash
sudo pacman -S pywal
```

2. 生成配色：

隨便選一張你的桌布圖片路徑執行：

```bash
wal -i /path/to/your/wallpaper.jpg
```

執行後，pywal 會自動在 `~/.cache/wal/` 下生成一堆檔案，其中就包括 Waybar 需要的 `colors-waybar.css`。
